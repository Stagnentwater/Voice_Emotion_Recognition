Core conceptsKeep these in mind as we go:
1. Signal basics: audio = array of floats; sample rate = samples/second; mono = single channel.
2. Preprocessing: resample to 16kHz, convert stereo→mono (average channels).
3. Inference: a pretrained HuBERT/Wav2Vec2 model does feature extraction internally; you supply audio+rate.
4. Interpretation: model outputs labels + scores; scores ≠ probabilities in a strict sense, but are confidence-like.
5. Limitations: actors’ datasets ≠ real-world accents/noise; results can be uncertain or biased.

For step 2, how to handle audi in inside the app using the simple & quick way with streamlit-audiorec
1. packages needed: sreamlit-audiorec, soundfile, numpy

from st_audiorec import st_audiorec
import streamlit as st
import io, soundfile as sf
import numpy as np

st.subheader("🎤 Record your voice")
wav_audio = st_audiorec()   # UI button appears

if wav_audio is not None:
    # Playback right away
    st.audio(wav_audio, format="audio/wav")

    # Convert WAV bytes -> numpy waveform + sample rate
    data, sr = sf.read(io.BytesIO(wav_audio), dtype="float32", always_2d=False)
    # If stereo, average to mono now or later:
    if data.ndim > 1:
        data = data.mean(axis=1)

    st.success(f"Captured {len(data)/sr:.2f}s at {sr} Hz")
